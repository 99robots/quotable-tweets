== Description ==

The Quotable Tweets plugin by https://99robots.com gives you an easy way to add a beautiful actionable tweet link to your sidebar.

Please check out our other [plugins](https://99robots.com/products/?utm_source=wprepo&utm_medium=quotable-tweets&utm_campaign=desc) :)

Contribute to the plugin and submit pull requests through the [Quotable Tweets plugin for WordPress GitHub repository](https://github.com/wpsite/quotable-tweets).

== Installation ==

1. Upload `quotable-tweets` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
